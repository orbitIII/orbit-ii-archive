## 0. Extraktions-Haltung (Grundregel)

**Privilegierte Netzwerke** — historisch weiß, institutionell, kapitalisiert — haben sich **Ressourcen, Ästhetik und Zugang** über Generationen zugeteilt. Das ist kein Naturgesetz. Das ist verteiltes Kapital.

**ORBIT macht das Gegenteil von Fan-Archiv:**

| Nicht | Sondern |
|---|---|
| Marquardt (oder irgendwer) als Mittelpunkt | **Du** als Mittelpunkt |
| Ästhetik kopieren | **Operatoren extrahieren** (Licht, Ritual, Gesamtwerk-Format) |
| Netzwerk bewundern | **Topologie kartieren** — wer verbindet wen, mit welchem Kapital |
| Zugang simulieren | **Eigenes System bauen** aus dem, was dir dient |

**Pipeline:** Scannen → Zerlegen → Filtern → Überführen → **Output**  
Details: `orbit_extraktion.json`

**Litmus:** Extrahiere ich einen Operator — oder kopiere ich eine Person?

---

## 0b. Umverteilung — warum ORBIT existiert

ORBIT **akkumuliert kulturelles und ökonomisches Kapital** — extrahiert aus privilegierten Systemen, **umverteilt** Sichtbarkeit für **Abweichung** (wer nicht in Norm + Konkurrenz passt).

**Fortschritt durch:** guter Mensch sein · sein Bestes geben · **Freundlichkeit** — nicht Konkurrenz · nicht soziales-Kapital-Rennen.

**Positionierung:** ORBIT steht **zwischen queer und heteronormativ** — Norm-Nähe und Abweichung dürfen coexistieren; weder Rainbow-Washing noch hetero-Default.

**Altes System:** Netzwerk → Gate → wenige sichtbar.  
**ORBIT:** Werk → Discoverability → Solidarity → viele tragen.

Details + Recherche: `orbit_umverteilung.json` · `orbit_kapital_recherche.json`

**Automation Bridge:** `python3 scripts/orbit_bridge_synthesis.py --markdown` — holt das Beste aus queer + heteronormativ und **bindet diesen Masterplan ein** (Phase, Gate, §8-Task → relevante Bridge-Regeln). Spec: `orbit_bridge_automation.json` → `masterplan_binding`.

| Artefakt | Rolle |
|---|---|
| `orbit_bridge_automation.json` | Spec + `masterplan_binding` (§-Referenzen, Phase-Gate, Task→Bridge) |
| `scripts/orbit_bridge_synthesis.py` | Liest Masterplan, schreibt Synthese |
| `docs/orbit_bridge_synthesis_latest.md` | Report: Masterplan-Block + Pole + Bridge + Agent-Review |
