# Orbit II Core Questions

Orbit II is not a database.

Orbit II exists to answer musical questions that are difficult or impossible to answer manually.

Every dataset, automation and analysis should ultimately contribute to answering one or more of these questions.

---

## Digging

- Which tracks are repeatedly discovered independently by Panorama Bar artists?
- Which labels consistently produce tracks that survive over multiple years?
- Which shops repeatedly expose future Panorama Bar material before it becomes widely known?
- Which releases connect otherwise unrelated artists?

---

## Musical Context

- Why is a track played?
- At which point of a set is it typically used?
- Which tracks frequently appear before or after it?
- Which atmosphere does it create?
- Which musical function does it fulfil?

---

## Collection Management

- Do I already own similar tracks?
- Is this adding something genuinely new?
- Which Rekordbox folder should this belong to?
- Is this a Flow, Push, Layer, Atmosphere or Moment track?
- Which tracks are currently underrepresented in my library?

---

## Set Preparation

- Build a warm-up trajectory.
- Build a Panorama Bar opening.
- Build a closing section.
- Find bridge tracks between two musical worlds.
- Find tracks that maintain tension without increasing energy.

---

## Research

- How has Panorama Bar changed between 2021-2026?
- Which artists became central?
- Which communities emerged?
- Which labels gained or lost influence?
- Which musical characteristics became more common over time?

---
